# EasyTranslator: Простая установка перевода для игр

**EasyTranslator** - программа, которая позволит вам установить в игру локализацию с языком, которого официально нет в самой игре. От вас не требуется заходить и искать каждый раз перевод для вашей игры. Единственное, что вам нужно сделать - запустить программу, выбрать игру и нажать на кнопку **Установить** и готово! Дальше программа сделает всё за вас.

## ⚙️ Как это работает

1. Запускаешь программу.
2. Вбиваешь в поиск название игры.
3. Выбираешь нужную иконку из списка.
4. Нажимаешь "Установить". Программа сама скачает и распакует перевод по нужному пути.

## 🏗 Простой интерфейс
Сначала вам будет предоставлен выбор из игр, перевод которых был недавно обновлён. Чтобы найти свою игру вам достаточно будет вбить в поиск название вашего проекта и выбрать иконку вашей игры.

## 🛠 Стэк разработки
Языки: Rust 1.94.1, JavaScript / TypeScript
Фреймворки: Tauri v2, React

## 🚀 Установка и запуск (Для разработчиков)
1. Установите Node.js и Rust.
2. Клонируйте репозиторий:
```bash
git clone https://github.com/AngeloDeveD/EasyTranslator.git
cd EasyTranslator
```
3. Установите зависимости и запустите:
```bash
npm install
npm run tauri dev
```

4. Для создания билда запустите:
```bash
npm run tauri build
```
